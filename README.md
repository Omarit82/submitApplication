# Submit Application App.

## Installation:

Clone repository:

`git clone https://github.com/Omarit82/submitapplication`

Navegate to ../submitapplication once inside the project folder execute:

To install all necessary dependencies:

`npm install`

To run de project:

`npm run dev`

The project will be available in http://localhost:5173/
